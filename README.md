## Install and Run

[Install dependencies][dependencies] to run your first Jekyll site. Then follow the instructions to run my site.

```bash
$ git clone https://github.com/shahnawaz-pabon/jekyll-site.git
$ cd jekyll-site
$ bundle install
$ bundle exec jekyll serve
```
